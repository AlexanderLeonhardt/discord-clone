## Project Purpose

Creating this project to learn something from recreating Discord's design and learn TypeScript along the way.

## Command Cheat Sheet

```bash
npm run dev
npm run build
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
